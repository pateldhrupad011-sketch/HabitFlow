# HabitFlow
HabitFlow is a habit tracker built with Python and HTML/CSS/JavaScript. Add daily habits in Health, Study, and Fitness categories, check them off to build streaks, and view a 7-day progress grid. Get reminders for unfinished habits with auto-saving. Features a dark UI with stat cards, animations, and motivational quotes on launch.
