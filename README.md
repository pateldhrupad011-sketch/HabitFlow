# HabitFlow
HabitFlow is a habit tracker built with Python and HTML/CSS/JavaScript. Add daily habits in Health, Study, and Fitness categories, check them off to build streaks, and view a 7-day progress grid. Get reminders for unfinished habits with auto-saving. Features a dark UI with stat cards, animations, and motivational quotes on launch.


This is the page of my website, designed in a dark theme. It includes three options to add a habit, and a fourth option to create a new habit if the one you want is not already listed. You can also view the total number of habits you have created, the habits you completed that day, your streak to help keep you motivated, and the percentage of habits you completed out of the ones you planned for the week. I hope you find it useful and use it in your daily life to improve yourself every day.
